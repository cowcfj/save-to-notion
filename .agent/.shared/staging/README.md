# Staging Area (暫存區)

本目錄 (`.agent/.shared/staging/`) 用於存放 Agent 協作過程中的**中間產物**與**臨時文件**。

## 用途範例

- **代碼生成中間態**: Agent 生成的大型重構草稿，待 Reviewer Agent 檢查。
- **臨時測試數據**: 針對特定 Bug 臨時生成的 JSON/CSV 檔案。
- **分析報告**: 代碼庫掃描報告或複雜度分析結果。

## 清理規則

- 本目錄內容應視為**易失性 (Volatile)**。
- 任務完成後，Agent 應負責清理自己產生的臨時文件。
- 不應將此目錄內容提交到版本控制系統 (建議加入 `.gitignore`)。
