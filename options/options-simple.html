🔧 Notion API 設置

使用 Notion Integration 的 API 金鑰進行授權。這是最穩定可靠的方式。

點擊這裡創建 Notion Integration

請設置 API 金鑰
從 Notion Integrations 頁面獲取
從 Notion 數據庫 URL 中獲取，格式如：notion.so/[workspace]/[database-id]

📖 設置說明

1. 創建 Notion Integration

  1. 訪問 Notion Integrations
  2. 點擊「New integration」
  3. 填寫名稱（如：Web Clipper）
  4. 選擇關聯的工作空間
  5. 點擊「Submit」
  6. 複製生成的 API Key

2. 獲取數據庫 ID

  1. 在 Notion 中打開要保存內容的數據庫
  2. 從瀏覽器地址欄複製 URL
  3. URL 格式：https://www.notion.so/workspace/DATABASE_ID?v=VIEW_ID
  4. DATABASE_ID 就是需要的數據庫 ID

3. 共享數據庫給 Integration

  1. 在 Notion 數據庫頁面點擊右上角「Share」
  2. 點擊「Invite」
  3. 搜索並選擇您創建的 Integration
  4. 點擊「Invite」完成共享
