備份與還原

備份包含的數據：

  • 標註數據：所有頁面的高亮標記（文字、顏色、位置）
  • 保存記錄：已同步到 Notion 的頁面狀態
  • 遷移數據：舊版本資料的暫存記錄

注意：API Key 和 Database ID 等配置會透過 Google 帳號自動同步，不包含在備份中。

還原備份將覆蓋現有的標註數據，建議先匯出當前數據。

使用指南

了解如何最有效地使用 Notion Smart Clipper

快速開始

1. 連接擴展

點擊擴展圖標 → Settings → 點擊「連接到 Notion」

2. 授權訪問

在 Notion 創建 Integration，複製 Secret Key 並保存

3. 開始使用

選擇目標數據庫，點擊保存按鈕即可剪藏！

核心功能

一鍵保存
智能提取文章內容、圖片與 icon
多色標註
支持 4 種顏色與 3 種樣式（背景/文字/底線）
快捷鍵支援
按 Ctrl+S (Mac: Cmd+S) 快速開啟標註工具
自定義模板
支持 {title}, {date} 等變數
狀態同步
圖標徽章即時顯示保存狀態

常見問題

查看完整指南
API Key 無法連接？
請確認 Token 格式正確（以 secret_ 開頭），並確保網路連接正常。
數據庫列表為空？
請在 Notion 頁面右上角的三點菜單中，找到 "Connections" 並添加您的 Integration。
標註沒有顯示？
請嘗試刷新頁面。此功能需要 Chrome 105+ 支持。
