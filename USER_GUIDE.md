## 🚀 快速開始 (只需 2 分鐘)

只需三個簡單步驟，即可開始將網頁與靈感收集到您的 Notion 庫中。

### 第 1 步：安裝擴充功能

1. 前往 [Chrome Web Store - Save to Notion Smart Clipper](https://chromewebstore.google.com/detail/save-to-notion-smart-clip/gmelegphcncnddlaeogfhododhbcbmhp) 網頁。
2. 點擊「**加到 Chrome**」並確認新增。
3. _建議：將擴展圖標固定在瀏覽器工具列上，方便隨時點擊。_

### 第 2 步：連接 Notion 帳號 (提供兩種方式)

**👉 方式 A：快捷授權 (推薦，支援 OAuth 一鍵登入)**

1. 點擊瀏覽器右上角的 Notion Smart Clipper 圖標，進入 `⚙️ 設定`。
2. 點擊「**🔌 連接到 Notion**」按鈕，跳轉至 Notion 官方授權畫面。
3. 選擇您欲保存網頁的工作區 (Workspace) 或特定頁面並同意授權，完成後即可關閉視窗。

**👉 方式 B：手動輸入 Integration Token (傳統方式)**

1. 前往 [Notion Integrations](https://www.notion.so/my-integrations) 建立新的整合方案 (可命名為 "Chrome Clipper")。
2. 複製產生出來的 **Internal Integration Token** (`secret_...`)。
3. 進入擴展 `⚙️ 設定`，將您的 Token 貼至輸入框並點擊連接。
4. **【重要】**進入您的 Notion 目標資料庫頁面，點擊右上角 `...` →「**Connections**」→ 搜尋並加入您剛才創建的 Integration，確保擴展有權限存取。

### 第 3 步：選擇目標資料庫 (Database)

1. 回到擴展的設定介面，此時下拉選單中應會自動載入您已經授權的資料庫。
2. 選擇一個為本次目標的資料庫（Data Source）。
3. 點擊「**保存設置**」。

🎉 **設定完成！您現在可以開始盡情收集網頁了。**

> 遇到下拉選單找不到資料庫的情況？請參考文末的 [附錄：手動獲取 Notion ID 教學](#-附錄手動獲取-notion-id-教學)。

## 🎯 核心功能與使用情境

### 情境一：完美保存一篇文章

當您看到一篇想稍後閱讀或存檔的好文時：

1. **一鍵保存**：點擊擴展圖標，按下「**💾 保存頁面**」。
2. **狀態確認**：等待約 2-5 秒，擴展圖標會出現**綠色 "✓" 徽章**，代表保存成功。
3. **快速跳轉**：點擊彈出視窗上的「**🔗 在 Notion 中打開**」，即可直接在 Notion 中查看排版精美的文章。

> 💡 **小提醒**：對於某些需滾動才會加載內容的動態網頁，建議先「滑到底部」再進行保存，確保內容完整提取。

### 情境四：從 Popup 直接登入 Google 帳號

如果您要啟用 Google 帳號同步與進階功能，不必先進入完整設定頁：

1. 點擊瀏覽器工具列中的擴展圖示打開彈出視窗。
2. 點擊彈出視窗右上角顯示「登入」或「已登入」的帳號按鈕。
3. 未登入時會開啟 Google 登入流程；已登入時會直接跳到設定頁的「**進階功能**」區塊，管理帳號與雲端同步功能。

### 情境二：使用螢光筆標註重點

不想保存整篇文章，只想摘錄金句？或是邊看邊畫重點？

1. **開啟標註模式**：點擊擴展圖標，選擇「**🎨 開始標註**」。
2. **選取重點**：使用滑鼠選取網頁上的文字，會自動顯示五色螢光筆供您挑選：
   - 🟡 **黃色**：一般重點
   - 🟢 **綠色**：關鍵結論 / 實用解答
   - 🔴 **紅色**：需要進一步查證 / 疑問
   - 🔵 **藍色**：參考資料
3. **同步到 Notion**：標註完成後，點擊工具列上的「**同步**」按鈕，這些標註就會自動匯總到您的 Notion 頁面中。
4. **刪除標註**：若畫錯了，只需「雙擊」該段標註文字即可快速取消，或按住 `Ctrl`/`Cmd` + 點選標註。

### 情境三：自定義保存標題

如果您有自己習慣的筆記命名格式，可以進入 `⚙️ 設定` 的「頁面模板設置」修改：

**支援的動態變量：**

- `{title}` - 網頁原標題
- `{date}` - 今天的日期（如 2024-03-15）
- `{domain}` - 網站域名（如 github.com）
- `{url}` - 網頁原始連結

**實用範例：**

- `[{domain}] {title}` ➡️ `[medium.com] 提升工作效率的 10 個方法`
- `📌 {date} | {title}` ➡️ `📌 2024-03-15 | 提升工作效率的 10 個方法`

## 💡 達人技巧 (Pro Tips)

1. **批量狀態檢查**：打開多個文章分頁並切換時，不用每個都打開擴展面板。只要看擴展圖標右上角是否有綠色 "✓"，就能立刻辨識該網頁是否已存入 Notion。
2. **極速快捷鍵**：在瀏覽器網址列輸入 `chrome://extensions/shortcuts`，為本擴展設定喚醒快捷鍵（例如 `Alt+S`）。如此一來，您就能實現「全鍵盤保存」的高效工作流！
3. **無痛數據管理**：當您在 Notion 中刪除了已保存的頁面，擴展會在背景自動清理對應的網頁保存紀錄（本地標注仍會保留），避免佔用瀏覽器儲存空間。若需要深度清理，可至設定中的「數據管理」進行一鍵重整。

## ❓ 常見問題與故障排除

### 🔌 連線與設定

- **無法連接 Notion 或顯示 Invalid API Token？ (僅限方式 B)**
  如您是使用手動輸入 Token 的用戶，請確認 Token 複製時沒有混入多餘的空格。若仍無效，請至 Notion Integrations 重新生成一組新的 Token 替換。
- **下拉選單找不到目標資料庫？**
  請嘗試以下操作：
  1. **輸入名稱搜尋**：在下拉選單的輸入框內直接打字，搜尋您的資料庫名稱。
  2. **貼上 Notion 連結或 ID**：如果搜尋後仍找不到，請直接複製目標 Notion 頁面的網址（或傳入 Page ID / Database ID），並**貼入同一個下拉輸入框內**然後保存。_(獲取 ID 詳細方式請見 [附錄：手動獲取 Notion ID 教學](#-附錄手動獲取-notion-id-教學))_。

### 💾 保存與標註

- **保存失敗或顯示「Failed to fetch」？**
  這通常是因為寫入過於頻繁觸發了 API 限制（3次/秒），或是網路狀態不佳。請稍等片刻後重試。
- **點了「開始標註」但無法畫重點？**
  請確保您選取的文字屬於網頁的主體內容。在一些特殊的單頁應用程式（SPA）或是權限受限的頁面可能會受到限制。
- **內容保存不完整或缺漏？**
  如遇付費牆 (Paywall) 或需要登入後才能查看的內容，擴展將無法抓取。此外請確認頁面內容已經完全載入完畢再進行儲存。

### 🐞 如何取得錯誤日誌並回報？

若您遇到無法解決的異常錯誤：

1. 進入 `⚙️ 設定`。
2. 捲動到最下方的「**診斷工具**」，開啟「**啟用除錯日誌**」。
3. 點擊「**匯出偵錯日誌**」來下載 JSON 快照檔案。
4. 歡迎至 [GitHub Repository](https://github.com/cowcfj/save-to-notion) 提交 Issue，並附上日誌檔案供開發團隊分析排查。

## 📚 附錄：手動獲取 Notion ID 教學

<details>
<summary><strong>👉 展開查看：遇到下拉選單找不到資料庫時的手動作法</strong></summary>

偶爾因權限設定遺漏或延遲，選單可能未顯示您的目標資料庫。在問題排除前，您可以使用手動輸入 ID 作為備用方案。

### 如何獲取 Database ID / Page ID

1. **獲取 Database ID**：在 Notion 以全頁面模式打開目標數據庫。點擊右上角「Share」並選「Copy link」。連結中 `/` 與 `?v=` 之間的 32 字符即為 ID（如 `1234567890abcdef1234567890abcdef`）。
2. **獲取 Page ID**：打開特定 Notion 頁面，複製分享連結最後的 32 個字符即為 ID。

獲得 ID 或完整網址後，在擴展設定的「**Notion Data Source ID**」輸入框貼上該 ID，並點擊保存。
