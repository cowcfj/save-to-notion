備份與還原

備份包含的數據：

  • 標註數據：所有頁面的高亮標記（文字、顏色、位置）
  • 保存記錄：已同步到 Notion 的頁面狀態
  • 遷移數據：舊版本資料的暫存記錄

注意：API Key、Database ID 與 OAuth 連線資訊會透過本機或 Google 帳號重新建立，不包含在備份中。

還原備份將覆蓋現有的標註數據，建議先匯出當前數據。

使用指南

了解如何最有效地使用 Notion Smart Clipper

快速開始

1. 連接到 Notion

在「一般設定」點擊「以 OAuth 連接 Notion」授權，或選擇手動配置輸入 API Key。

2. 預備資料庫

確保您的 Notion 中已有資料庫，且該資料庫已授權給本擴展存取。

3. 開始剪藏

在任意網頁點擊擴展圖標，選擇目標資料庫後即可一鍵保存！

常見問題

查看完整指南
  • API Key 無法連接？
    請確認 Token 格式正確（以 secret_ 開頭），並確保網路連接正常。
  • 數據庫列表為空？
    請在 Notion 頁面右上角的三點菜單中，找到 "Connections" 並添加您的 Integration。
  • 標註沒有顯示？
    請嘗試刷新頁面。此功能需要 Chrome 105+ 支持。
