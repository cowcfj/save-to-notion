🎯 快速開始

  1. 點擊擴展圖標 → Settings
  2. 點擊「連接到 Notion」按鈕
  3. 在 Notion 中創建 Integration 並複製 API Key
  4. 選擇數據庫並保存設置
  5. 開始保存網頁！

� 智能數據管理

全新的數據監控和管理功能：

  • 實時監控：即時顯示存儲使用量和容量狀態
  • 智能優化：自動清理重複數據，提升效能
  • 數據備份：完整備份和恢復機制，確保數據安全
  • 完整性檢查：自動驗證數據完整性

💡 在設置頁面找到「數據管理」選項卡體驗新功能！

�🖼️ 智能圖片提取

強化的圖片處理功能，支援更多格式：

  • 懶加載圖片（data-src、data-lazy-src 等）
  • 響應式圖片（srcset）
  • 代理URL：自動處理新聞網站的代理圖片鏈接
  • URL清理：移除重複參數，確保圖片正常載入
  • 自動過濾小尺寸圖標
  • 支持各種 CMS 系統

🎨 模板自定義

靈活的內容格式化選項：

標題模板變量

  • {title} - 原始標題
  • {date} - 當前日期 (2024-01-15)
  • {time} - 當前時間 (14:30)
  • {datetime} - 完整時間 (2024-01-15 14:30)
  • {url} - 完整網址
  • {domain} - 網站域名

模板範例

  • [{domain}] {title} → [example.com] 文章標題
  • {title} - {date} → 文章標題 - 2024-01-15
  • {title} ({datetime}) → 文章標題 (2024-01-15 14:30)

額外選項：

  • ✅ 添加時間戳 - 在內容開頭顯示保存時間
  • ✅ 添加來源信息 - 在內容末尾添加原文鏈接

⚙️ 設置技巧

  • API Key 測試：輸入後點擊「測試 API Key」驗證連接
  • 自動載入數據庫：有效的 API Key 會自動顯示可用數據庫
  • 模板預覽：設置模板後點擊「預覽效果」查看結果
  • 數據庫權限：記得在 Notion 中將 Integration 連接到數據庫

🎨 跨元素文本標記功能

v2.4.6 全新功能：支援跨越多個HTML元素的文本標註，特別針對列表、段落和複雜結構優化。

💡 使用技巧

對於複雜的跨元素選擇（如跨多個列表項）：

  1. 先進行單元素標註：在目標區域內選擇並標記幾段單獨的文本
  2. 改變頁面結構：單元素標註會稍微調整DOM結構
  3. 進行跨元素選擇：現在可以成功選擇跨越多個元素的文本進行標記

🎯 適用場景

  • 列表標註：跨越多個 <li> 項目的內容
  • 段落標註：跨越多個 <p> 段落的文本
  • 引用標註：跨越 <blockquote> 邊界的選擇
  • 表格標註：跨越表格單元格的內容

🔬 技術原理：

系統使用6層漸進式回退機制處理複雜的DOM結構：

  • 標準DOM操作 → 內容提取 → 克隆重建 → 文本替換 → 片段重建 → 超級安全模式

🖱️ 標記刪除功能 (v2.4.8 全新優化)

雙重刪除系統：提供兩種靈活的標記刪除方式，滿足不同使用場景的需求。

🎯 方式一：雙擊刪除（推薦日常使用）

  1. 雙擊標記：雙擊任何已標記的文本
  2. 確認刪除：彈出確認對話框
  3. 完成刪除：點擊「確定」完成刪除

✨ 優點：有確認步驟，防止誤操作，操作直觀

⚡ 方式二：Ctrl/Cmd + 點擊（快速刪除）

  1. 按住快捷鍵：按住 Ctrl(Windows) 或 Cmd(Mac)
  2. 點擊標記：點擊要刪除的標記文本
  3. 立即刪除：標記立即被移除（無需確認）

⚡ 優點：操作快速，適合批量整理標記

💡 使用建議

  • 日常閱讀：使用雙擊刪除，安全可靠
  • 批量整理：使用 Ctrl/Cmd + 點擊，效率更高
  • 正常操作：普通點擊和文本選擇完全不受影響
  • 跨平台：Windows 用 Ctrl，Mac 用 Cmd，自動適配

🔧 v2.4.8 修復內容：

  • ✅ 徹底修復雙擊時只選中部分字符的問題
  • ✅ 解決雙擊刪除功能失效的問題

�🔧 常見問題與解決方案

  • 「Invalid image url」錯誤：已修復，會自動清理無效圖片URL
  • 保存失敗：檢查 API Key 和數據庫權限，確保Integration已連接
  • 圖片不顯示：某些網站有防盜鏈保護，擴展會自動跳過無效圖片
  • 內容不完整：嘗試在不同網站測試，某些網站結構特殊
  • 模板不生效：確保已保存設置並重新載入頁面
  • 存儲空間不足：使用「數據管理」功能進行空間優化

🎯 文本標記問題排解

  • 跨元素選擇失敗：先進行幾次單元素標註，再嘗試跨元素選擇
  • 列表項標記問題：先標記單個列表項，再嘗試跨項標記
  • 標記位置不準確：檢查控制台日誌，系統會自動選擇最佳回退方案
  • 標記無法點擊：雙擊標記文本可以刪除，確認事件監聽器正常
  • 調試信息：打開開發者工具查看詳細的處理過程和錯誤信息

💡 使用技巧

  • 🎯 最佳實踐：定期使用數據優化功能保持最佳性能
  • 🔄 升級安全：所有數據在升級過程中自動保留，無需擔心資料遺失
  • 🌐 網站相容性：支援主流新聞網站、部落格和技術文檔網站
  • 📱 跨平台：保存的內容可在所有Notion平台上查看和編輯
