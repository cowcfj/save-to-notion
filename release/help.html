📚 Notion Smart Clipper v2.5.3 使用指南

🎉 最新更新 v2.5.3

  • 🎨 多顏色標註選擇器：支援 4 種顏色（黃/綠/藍/紅）標註，可隨時切換
  • 🖱️ 雙重刪除機制：Ctrl/Cmd+點擊快速刪除 + 管理列表刪除
  • ✨ CSS Highlight API：全新的標註引擎，無需修改 DOM 結構
  • 🔄 無縫自動遷移：自動從舊版標註遷移到新版本
  • ⚡ 跨元素標註增強：完美支援跨越多個 HTML 元素的文本選擇

🎯 快速開始

  1. 點擊擴展圖標 → Settings
  2. 點擊「連接到 Notion」按鈕
  3. 在 Notion 中創建 Integration 並複製 API Key
  4. 選擇數據庫並保存設置
  5. 開始保存網頁！

📦 智能數據管理

全新的數據監控和管理功能：

  • 實時監控：即時顯示存儲使用量和容量狀態
  • 智能優化：自動清理重複數據，提升效能
  • 批量操作：快速清理過期或無效數據
  • 數據備份：支援導出和導入功能

📝 靈活的標題模板

使用變數自定義頁面標題格式：

可用變數

  • {title} - 網頁標題
  • {date} - 日期 (2024-01-15)
  • {time} - 時間 (14:30)
  • {datetime} - 完整時間 (2024-01-15 14:30)
  • {url} - 完整網址
  • {domain} - 網站域名

模板範例

  • [{domain}] {title} → [example.com] 文章標題
  • {title} - {date} → 文章標題 - 2024-01-15
  • {title} ({datetime}) → 文章標題 (2024-01-15 14:30)

額外選項：

  • ✅ 添加時間戳 - 在內容開頭顯示保存時間
  • ✅ 添加來源信息 - 在內容末尾添加原文鏈接

⚙️ 設置技巧

  • API Key 測試：輸入後點擊「測試 API Key」驗證連接
  • 自動載入數據庫：有效的 API Key 會自動顯示可用數據庫
  • 模板預覽：設置模板後點擊「預覽效果」查看結果
  • 數據庫權限：記得在 Notion 中將 Integration 連接到數據庫

🎨 多顏色文本標註功能

v2.5.3 全新升級：使用 CSS Highlight API 實現高性能、多顏色的文本標註系統。

💡 如何創建標註

  1. 選擇文本：用滑鼠選擇要標註的文本（支援跨元素選擇）
  2. 選擇顏色：在彈出的工具欄中點擊顏色按鈕
  • 🟨 黃色 - 預設顏色，適合一般重點標註
  • 🟩 綠色 - 適合標記正面內容、重要定義
  • 🟦 藍色 - 適合標記參考資料、補充說明
  • 🟥 紅色 - 適合標記警告、關鍵問題
  1. 創建標註：點擊「✨ 標註」按鈕，文本將以所選顏色高亮顯示
  2. 切換顏色：可以隨時點擊其他顏色按鈕切換標註顏色
  3. 同步到 Notion：點擊「💾 同步到 Notion」保存所有標註

🗑️ 如何刪除標註

方法一：快速刪除（推薦）

  • 按住 Ctrl（Windows/Linux）或 Cmd（Mac）
  • 點擊要刪除的標註文本
  • 標註會立即消失，無需確認

方法二：列表管理

  • 點擊工具欄中的「📋 管理」按鈕
  • 在彈出的標註列表中找到要刪除的項目
  • 點擊該項目右側的「🗑️ 刪除」按鈕

🔬 技術亮點：

  • CSS Highlight API：瀏覽器原生支援，無需修改 DOM 結構
  • 完美跨元素：可以標註任意複雜的跨元素文本選擇
  • 自動持久化：標註數據自動保存，刷新頁面後自動恢復
  • 無縫遷移：自動從舊版標註系統遷移，不丟失任何數據
  • 高效渲染：使用瀏覽器原生 API，性能優於傳統方案

�� 使用建議：

  • 使用黃色標記一般重點內容
  • 使用綠色標記重要定義、關鍵概念
  • 使用藍色標記補充說明、參考資料
  • 使用紅色標記需要特別注意的警告、問題
  • 善用 Ctrl/Cmd+點擊 快速刪除不需要的標註
  • 定期使用「💾 同步到 Notion」保存標註到雲端

💡 常見問題與解決方案

  • 標註沒有顯示：需要 Chrome 105+ 或 Edge 105+；嘗試刷新頁面
  • 刪除標註無效：確認正確按住 Ctrl/Cmd 鍵再點擊；或使用「📋 管理」列表刪除
  • 舊版標註遷移：系統會自動檢測並遷移，不會丟失數據
  • API Key 無法連接：確認在 Notion 中正確創建 Integration 並連接到數據庫
  • 保存失敗：檢查 API Key 和數據庫權限
  • 圖片不顯示：某些網站有防盜鏈保護，擴展會自動跳過
  • 存儲空間不足：使用「數據管理」功能進行優化
