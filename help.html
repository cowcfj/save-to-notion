🎉 最新更新 v2.7.3

  • 🐛 超長文章支持：完整保存任意長度的文章，自動分批處理
  • 🎯 圖標徽章顯示：擴展圖標上顯示綠色 "✓" 表示頁面已保存
  • 🔗 Open in Notion：保存成功後一鍵打開對應的 Notion 頁面
  • 🎨 多顏色標註：支援 5 種顏色（黃/綠/藍/紅/紫）標註

🎯 快速開始 (3 步驟)

  1. 點擊擴展圖標 → Settings → 「連接到 Notion」
  2. 在 Notion 創建 Integration，複製 API Key 並貼上
  3. 選擇數據庫 → 開始保存網頁！

⚡ 核心功能

  • 一鍵保存：智能提取文章內容、圖片、圖標
  • 多顏色標註：使用 5 種顏色標記重點內容
  • 自定義模板：使用變數 ({title}, {date}, {domain}) 自定義標題格式
  • 圖標徽章：擴展圖標顯示綠色 ✓ 表示頁面已保存
  • Open in Notion：保存後一鍵打開對應頁面
  • 超長文章：自動分批處理，支持任意長度文章

💡 Top 10 常見問題

  1. API Key 無法連接? → 確認 Token 格式正確 (以 secret_ 開頭)
  2. 數據庫列表為空? → 在 Notion 中添加 Integration Connection
  3. 保存失敗? → 檢查 API Key 和數據庫權限
  4. 內容不完整? → 等待頁面完全加載後再保存
  5. 標註沒有顯示? → 需要 Chrome 105+，嘗試刷新頁面
  6. 無法刪除標註? → 按住 Ctrl/Cmd 點擊，或使用「📋 管理」功能
  7. 圖標徽章不顯示? → 確認已保存頁面，刷新後查看
  8. "Open in Notion" 按鈕不顯示? → v2.7.0+ 頁面需要，舊頁面需重新保存
  9. 圖片不顯示? → 某些網站有防盜鏈保護
  10. 存儲空間不足? → 設置頁面使用「數據管理」清理
