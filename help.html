📚 Notion Smart Clipper 使用指南

🎯 快速開始

  1. 點擊擴展圖標 → Settings
  2. 點擊「連接到 Notion」按鈕
  3. 在 Notion 中創建 Integration 並複製 API Key
  4. 選擇數據庫並保存設置
  5. 開始保存網頁！

🖼️ 智能圖片提取

自動識別和提取網頁中的圖片，支持：

  • 懶加載圖片（data-src、data-lazy-src 等）
  • 響應式圖片（srcset）
  • 自動過濾小尺寸圖標
  • 支持各種 CMS 系統

🎨 模板自定義

靈活的內容格式化選項：

標題模板變量

  • {title} - 原始標題
  • {date} - 當前日期 (2024-01-15)
  • {time} - 當前時間 (14:30)
  • {datetime} - 完整時間 (2024-01-15 14:30)
  • {url} - 完整網址
  • {domain} - 網站域名

模板範例

  • [{domain}] {title} → [example.com] 文章標題
  • {title} - {date} → 文章標題 - 2024-01-15
  • {title} ({datetime}) → 文章標題 (2024-01-15 14:30)

額外選項：

  • ✅ 添加時間戳 - 在內容開頭顯示保存時間
  • ✅ 添加來源信息 - 在內容末尾添加原文鏈接

⚙️ 設置技巧

  • API Key 測試：輸入後點擊「測試 API Key」驗證連接
  • 自動載入數據庫：有效的 API Key 會自動顯示可用數據庫
  • 模板預覽：設置模板後點擊「預覽效果」查看結果
  • 數據庫權限：記得在 Notion 中將 Integration 連接到數據庫

🔧 常見問題

  • 保存失敗：檢查 API Key 和數據庫權限
  • 圖片不顯示：某些網站有防盜鏈保護
  • 內容不完整：嘗試在不同網站測試，某些網站結構特殊
  • 模板不生效：確保已保存設置並重新載入頁面
